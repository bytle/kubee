## NAME

Generate the k8s manifest from jsonnet files.

This script uses arg $1 (name of *.jsonnet file to use) to generate the manifests/*.yaml files.

## SYNOPSIS

Usage:

```bash
kubee-jsonnet file.jsonnet
```
